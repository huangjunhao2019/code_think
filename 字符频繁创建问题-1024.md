```c++
class Solution {
public:
    string removeDuplicates(string s) {
        stack<char> st;
        for(int i=0;i<s.size();i++){
            if(st.empty()||st.top()!=s[i]){
                st.push(s[i]);
            }
            else{
                st.pop();
            }
        }
        string result="";
        while(!st.empty()){
            result=st.top()+result;
            st.pop();
        }
      //  reverse(result.begin(),result.end());
        return result;
    }
};
```

发现采用上述代码会出现超时问题，主要问题在与下面的

```cpp
 result=st.top()+result;
```

如果修改成为

```cpp
result=result+st.top();
```

就可以在规定时间内完成。

这说明了第一个用时比第二个长，第一个应该会频繁创建对象。
